ntopng - Web-based Network Monitoring

ntopng is the next generation version of the original ntop, a network traffic probe that monitors network usage. ntopng is based on libpcap/PF_RING and it has been written in a portable way in order to virtually run on every Unix platform, MacOS and on Windows as well.

In this video, we will look at how ntopng can be used to perform behavioral traffic analyses, to uncover suspicious or unwanted network activities.

Here we will look at how ntopng can be used also to actively monitor hosts and services, to timely react to potential outages and security risks.

ntopng Community and Bug Reporting

To report bugs you can use the official GitHub ntopng issues page. To say in touch with the community and the developers, you can join public ntop discussions or the FOSDEM21 chatroom.

Help steer ntopng, your opinion matters

ntopng has historically relied on the community to grow and evolve. The easiest way to contribute to ntopng is to try out our development releases. Offer us feedback as soon as possible so we can spot bugs early and improve quickly.
